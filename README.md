# learningJava
Aprendendo java

A meta de **1 commit por dia** é excelente. O segredo não é fazer projetos gigantes, mas uma progressão que te obrigue a relembrar conceitos gradualmente. Como você já programa em Python e C#, eu focaria em Java puro primeiro, sem Spring.
